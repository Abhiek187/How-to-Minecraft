## Commands

**Generate Data**

```bash
gradlew runData
```

**Run Client**

```bash
gradlew runClient
```

**Run Server**

```bash
gradlew runServer
```

**Run Game Test Server**

```bash
gradlew runGameTestServer
```

**Check for Dependency Updates**

```bash
gradlew dependencyUpdates
```
